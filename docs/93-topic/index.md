# 93. Регулярные Выражения / Regular Expressions - regex

- Специальные символы / Special Characters - ., *, +, ?, ^, $
- Классы символов / Character Classes - \d, \w, \s
- Квантификаторы / Quantifiers - {n,m}
- Группы / Groups - ()
- Несохраняющие группы / Non-capturing Groups - (?:)
- Позитивный/Негативный просмотр вперед/назад / Lookahead/Lookbehind - (?=), (?!), (?<=), (?<!)
- Флаги / Flags - re.I, re.M, re.S
